Core Pattern: Closed-Loop Validation

# Iterative Refinement Loop - Code Generator Example
for iteration in range(MAX_ITERATIONS):
    # Plan → Research → Execute
    tasks = planner.run(goal)
    research = researcher.run(tasks)
    result = executor.run(research)
    
    # Validate quality
    critique = critic.run(result)
    
    if critique["pass"]:
        break  # Quality gate passed
    
    # Refine with structured feedback
    goal = f"{goal}\n\nFeedback: {critique['feedback']}"

Generation → Validation → Repair cycle prevents incomplete/broken outputs

RAG Context Injection

def retrieve_memory(query_text):
    """Query vector store for relevant context"""
    query_vector = embedding_model.encode(query_text, normalize_embeddings=True)
    
    results = []
    for mem in memory_store:
        score = cosine_similarity(query_vector, np.array(mem['vector']))
        results.append((score, mem['text']))
    
    results.sort(key=lambda x: x[0], reverse=True)
    relevant = [text for score, text in results if score > 0.40][:3]
    
    return "\n".join(relevant)  # Inject into agent prompts

Semantic search improves factual grounding under token/latency constraints
